Animated Focus outline

Press the [tab] key or click on inputs to move focus from one element to the next.
The focus element speed is regulated by a parameter (in pixel per second) but there's also a max-speed parameter (in ms) so that the element doesn't move too quickly between two close elements.
For elements with already very evident focus styles, such as a button, I added a "data-nofocustyle" attribute so that the focus outline just fades out.
